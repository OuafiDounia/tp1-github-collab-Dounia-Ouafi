# Contributing Guide

## Nommage des branches
- feat/... pour nouvelles fonctionnalités
- fix/... pour corrections
- chore/... pour tâches annexes

## Convention de commit
- feat: description
- fix: description
- chore: description

## Processus
1. Fork/Clone le repo
2. Créer une branche depuis `develop`
3. Commit les modifications
4. Push et créer PR
5. Reviewer valide la PR
