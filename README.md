# Hello World Node.js Project

## Description
Ce projet est un petit serveur Node.js utilisant Express qui affiche "Hello World" sur la page d'accueil. Il sert de base pour apprendre à créer des applications web avec Node.js et Express.

## Technologies utilisées
- Node.js
- Express
- Git

## Instructions d’installation
1. Cloner le projet :

git clone <URL_DU_PROJET>
cd hello-node
Installer les dépendances :

npm install
Lancer le serveur :

node server.js
Ouvrir le navigateur et aller sur :

http://localhost:3000
